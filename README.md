# 12-bit CPU Design using Logisim

A custom-built **12-bit CPU architecture** designed and implemented using **Logisim** as part of the **Computer Architecture Laboratory (CSE 2114)** course at **Khulna University of Engineering & Technology (KUET)**.

This project demonstrates the fundamental concepts of processor design including datapath organization, ALU operations, register transfer, instruction execution, and control unit implementation using digital logic circuits.

---

## 📌 Project Overview

The primary objective of this project was to design a fully functional **12-bit processor** from scratch and simulate its operation using Logisim.

The processor supports:

- Arithmetic Operations
- Logical Operations
- Data Transfer Instructions
- Register Operations
- Memory Interaction
- Clock-Synchronized Execution

The architecture was implemented modularly to better understand the internal organization and workflow of a CPU.

---

## 🧠 CPU Components

### 🔹 Arithmetic Logic Unit (ALU)
The ALU performs several operations including:

- Addition
- Subtraction
- AND
- OR
- XOR
- NOT
- Shift Operations

### 🔹 Control Unit
Responsible for:

- Instruction Decoding
- Generating Control Signals
- Coordinating CPU Operations

### 🔹 Register File
Stores intermediate and operational data during instruction execution.

### 🔹 Memory Unit
Handles instruction and data storage for the processor.

### 🔹 Datapath
Connects all major CPU components and controls data movement throughout the system.
